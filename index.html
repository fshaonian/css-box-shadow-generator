How to Use the CSS Box Shadow Generator

This free tool lets you visually create CSS box shadows without writing code manually. Adjust the sliders to control the shadow's position, blur, spread, and opacity — then copy the generated CSS with one click.

You can stack multiple shadow layers to create complex effects like neumorphism, glow effects, or layered depth.

  • Horizontal / Vertical Offset — moves the shadow left/right or up/down
  • Blur Radius — higher values create a softer, more diffused shadow
  • Spread Radius — positive values expand the shadow, negative values shrink it
  • Inset — turns the shadow into an inner shadow
  • Multiple Layers — click "Add Layer" to stack shadows for richer effects
